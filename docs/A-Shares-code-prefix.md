Title: 科普股票字母标识大全：20种不同股票字母标识的意思

URL Source: https://zhuanlan.zhihu.com/p/21950307978

Markdown Content:
投资者在进行股票投资时，经常会遇到一些以特定字母开头的股票，例如C、N、U、W、V等，或者股票代码后带有特殊字母。今天，就带大家了解20种不同股票字母标识的意思。

**1、N，**新股上市首日的名称前都会加一个字母N，即英文new的意思，首日之后标识取消。

**2、C**，表示该股票是上市第2-5日，第6日开始标识取消。

**3、R，**表示该股票可以进行[融资融券](https://zhida.zhihu.com/search?content_id=253412163&content_type=Article&match_order=1&q=%E8%9E%8D%E8%B5%84%E8%9E%8D%E5%88%B8&zd_token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJ6aGlkYV9zZXJ2ZXIiLCJleHAiOjE3ODI0NjYzMjMsInEiOiLono3otYTono3liLgiLCJ6aGlkYV9zb3VyY2UiOiJlbnRpdHkiLCJjb250ZW50X2lkIjoyNTM0MTIxNjMsImNvbnRlbnRfdHlwZSI6IkFydGljbGUiLCJtYXRjaF9vcmRlciI6MSwiemRfdG9rZW4iOm51bGx9.OievWERE80n_ZZA-nLwRjSLnYaprw7RXKzstOJalyVo&zhida_source=entity)。

**4、kr**，意思是可融资融券的[科创板](https://zhida.zhihu.com/search?content_id=253412163&content_type=Article&match_order=1&q=%E7%A7%91%E5%88%9B%E6%9D%BF&zd_token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJ6aGlkYV9zZXJ2ZXIiLCJleHAiOjE3ODI0NjYzMjMsInEiOiLnp5HliJvmnb8iLCJ6aGlkYV9zb3VyY2UiOiJlbnRpdHkiLCJjb250ZW50X2lkIjoyNTM0MTIxNjMsImNvbnRlbnRfdHlwZSI6IkFydGljbGUiLCJtYXRjaF9vcmRlciI6MSwiemRfdG9rZW4iOm51bGx9.av0ZmtMwPVN1TD2UdsT_Ru1Q1UqF-x9-9RVqeLITgJg&zhida_source=entity)个股，k代表科创板，r代表融资融券。

**5、U，**是指在科创板上市，但尚未盈利的企业的股票。发行人首次实现盈利的，该特别标识取消。

**6、W，**是指在科创板上市，发行人具有表决权差异安排的，即同股不同权的股票。上市后不再具有表决权差异安排的，该特别标识取消。

**7、V，**是指在科创板上市，发行人具有[协议控制架构](https://zhida.zhihu.com/search?content_id=253412163&content_type=Article&match_order=1&q=%E5%8D%8F%E8%AE%AE%E6%8E%A7%E5%88%B6%E6%9E%B6%E6%9E%84&zd_token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJ6aGlkYV9zZXJ2ZXIiLCJleHAiOjE3ODI0NjYzMjMsInEiOiLljY_orq7mjqfliLbmnrbmnoQiLCJ6aGlkYV9zb3VyY2UiOiJlbnRpdHkiLCJjb250ZW50X2lkIjoyNTM0MTIxNjMsImNvbnRlbnRfdHlwZSI6IkFydGljbGUiLCJtYXRjaF9vcmRlciI6MSwiemRfdG9rZW4iOm51bGx9.CF48Ru4FoRBDfqk4sTPk6A1UWgTXiqx7DVZFrNWJiIY&zhida_source=entity)或者类似特殊安排的股票。上市后不再具有相关安排的，该特别标识取消。

**8、G，**代表股票对指数的贡献。“G”是指“贡”字，也就是点一下可以看见所有股票对该指数的涨跌贡献度。

**9、L，**股票名字前面“L”是指“联”，也就是表示此股存在关联品种，比如有B股，可转债，H股或权证等等，点击之可以切换到相关的品种。

**10、S，**S股在我国是指尚未进行股权分置改革或者已进入改革程序但尚未实施股权分置改革方案的股票。

**11、XR，**表示该股已除权，如果是送红股或者转股、配股后，股票总股本数增加，股票每股净资产有所减少，为了保持投资者资产大小不变，就会对股价进行除权，降低股价。

**12、XD**，表示股票除息，股票现金分红后，此时股票总数不变，但投资者已经获得部分股利，所以选择扣除掉股票中的股息来降低股价，保持投资者资产大小不变。

**13、DR**，表示[除权除息](https://zhida.zhihu.com/search?content_id=253412163&content_type=Article&match_order=1&q=%E9%99%A4%E6%9D%83%E9%99%A4%E6%81%AF&zd_token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJ6aGlkYV9zZXJ2ZXIiLCJleHAiOjE3ODI0NjYzMjMsInEiOiLpmaTmnYPpmaTmga8iLCJ6aGlkYV9zb3VyY2UiOiJlbnRpdHkiLCJjb250ZW50X2lkIjoyNTM0MTIxNjMsImNvbnRlbnRfdHlwZSI6IkFydGljbGUiLCJtYXRjaF9vcmRlciI6MSwiemRfdG9rZW4iOm51bGx9._RiwkHQ397n6Z4reBmmgZXPTw5y3DBrKiYnpRU5cGT0&zhida_source=entity)，股票既进行了现金分红又进行了送股或者配股，需要对分红股票进行除权除息使投资者总资产不变。

**14、ST，**ST是Special Treatment的缩写，意思为“特别处理”，一般表示该上市公司存在财务或其他异常状况，其股票交易需要进行特别处理。

**15、*ST，**比ST要严重，是股票[退市风险警示](https://zhida.zhihu.com/search?content_id=253412163&content_type=Article&match_order=1&q=%E9%80%80%E5%B8%82%E9%A3%8E%E9%99%A9%E8%AD%A6%E7%A4%BA&zd_token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJ6aGlkYV9zZXJ2ZXIiLCJleHAiOjE3ODI0NjYzMjMsInEiOiLpgIDluILpo47pmanorabnpLoiLCJ6aGlkYV9zb3VyY2UiOiJlbnRpdHkiLCJjb250ZW50X2lkIjoyNTM0MTIxNjMsImNvbnRlbnRfdHlwZSI6IkFydGljbGUiLCJtYXRjaF9vcmRlciI6MSwiemRfdG9rZW4iOm51bGx9.3J8ZmBdOjX-E-cNrGRWZ3xELsGjD_8coSnxlH9GJgUU&zhida_source=entity)，意味着这家上市公司面临着严重的财务或经营问题，有可能被交易所终止上市。

**16、PT，**PT是英文Particular Transfer（特别转让）的缩写，PT股就是停止任何交易，价格清零，等待退市的股票。

**17、NST，**经过重组或股改重新恢复上市的ST股。

**18、SZ**，是指在深圳证券交易所上市的股票。

**19、SH，**是指在上海证券交易所上市的股票。

**20、BJ，**是指在北京证券交易所上市的股票。
