## 策略概览

本策略基于**基本面+技术面相结合的多因子选股模型**，聚焦沪深两市主板、创业板、科创板及多个重要指数成分股。

### 选股逻辑

| 步骤 | 指标 | 条件 |
|---|---|---|
| 1. 股票池 | 沪深主板 + 创业板 + 科创板 + 重要指数成分股 | 剔除ST、停牌 |
| 2. 动量筛选 | 近5日成交量环比增长 | > 7%（较前5日） |
| 3. 涨幅过滤 | 近3日累计涨幅 | 0% ~ 17% |
| 4. 趋势排序 | 5日涨幅斜率排名 | 前20% |
| 5. 趋势确认 | 60日均线贴近度 | — |
| 6. 最终排序 | 市值得分 | 取前5只 |

### 持仓与调仓

- **持仓数量：** 5只，均等分配
- **调仓频率：** 每5个交易日（日频可配）
- **交易费用：** 固定单笔成本模型

### 风险控制

- 剔除ST及高风险标的
- 持仓数量限制（5只）
- 调仓频率限制
- 以沪深300为基准

## 与 ts_7AZ CANSLIM 对比

| 维度 | BigQuant 多因子动量 | iMobile ts_7AZ (CANSLIM) |
|---|---|---|
| **选股因子** | 成交量环比 + 涨幅斜率 + 均线 | C-A-N-S-L-I-M 7因子 |
| **侧重** | 技术动量为主 | 基本面+技术面均衡 |
| **持仓数** | 5只 | 动态（按regime: 5-12只） |
| **调仓频率** | 5天 | 每日重评估 |
| **风控** | ST剔除 + 持仓限制 | 4-regime TP/SL + 持有天数上限 |
| **适用** | 短期趋势 | 中长期质量+趋势 |
| **回测周期** | 15天（太短） | 170天+ |

### 可借鉴点

1. **成交量环比筛选**（5日 vs 前5日增长>7%）→ 可加入 ts_7AZ 的 I 因子（机构/换手率）增强
2. **涨幅斜率排名前20%** → 类似 RPS 排名思路，但使用更短窗口（5日 vs 60日）
3. **60日均线贴近度** → 可作为 M 因子（市场趋势）的细化指标
