# 同花顺(ths) vs 东方财富(dc) 概念板块

## Data Source by Tushare API

Fetch [打板专题数据](https://tushare.pro/document/2?doc_id=346) to get concept sector data of 东方财富dc & 同花顺ths, include 概念板块, 概念成分, 概念和行业指数行情.

## Main Differences

- 同花顺的概念板块更侧重于“技术挖掘”与“规范审核”, **偏向于基于公司基本面和公开信息做决策**
- 东方财富的“主题投资”则更注重“市场热度”与“资金驱动”, **短线交易者，对市场热点的嗅觉非常敏感**

| 对比维度 | 📈 同花顺 | 📊 东方财富 |
| :--- | :--- | :--- |
| **构建逻辑** | 基于**公开信息**与技术**模型初筛**，强调**客观与规范** | 综合**资讯、资金、情绪**，追踪**动态热点** |
| **数据来源** | 上市公司**互动易、公告、官微**等 | **财经新闻、政策、行业动态**、**股吧社区**讨论 |
| **更新机制** | **模型初筛** + **严格人工审核**，用户可反馈 | **算法实时捕捉**热点线索 + 编辑团队分析 |
| **风格特点** | **结构稳定**，数据**准确客观**，覆盖面广 | **灵活敏感**，**时效性强**，侧重资金与情绪 |


## Use Both Together

用**东方财富**来**发现和追踪当前市场的热点主题**，感受市场的温度；然后切换到**同花顺**，**仔细查验相关概念股是否被收录及其依据**，进行二次验证。这样既能把握先机，又能兼顾稳健.


## Solve Tushare API limit issues

- Max lines limit：单次最大3000行数据（需6000积分），可根据指数代码、日期参数循环提取.

- Points limit: replace by [AData API](https://github.com/1nchaos/adata)
